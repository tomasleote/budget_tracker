## Quick Start

### Install Dependencies
```bash
npm run install-all
```

### Development
```bash
# Start frontend (React app on http://localhost:3000)
npm run frontend

# Start backend (Node.js API on http://localhost:3001)
npm run backend

# Or start frontend with default script
npm start
```

### Build Production
```bash
npm run build
```
